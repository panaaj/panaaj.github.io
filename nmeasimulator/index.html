NMEASimulator is a FREE utilitiy for generating NMEA data streams to devices connected via TCP/IP, WebSocket or Serial port.

NMEASimulator is available for Windows, Mac, Linux & Raspberry Pi platforms and is a direct replacement for the NMEASimulator Chrome App (for which support ends as of June 2020 for these platforms).

It is a simple and easy to use tool that mimics vessel movement, engine status, water depth, etc.
Just open NMEASimulator and click on play_arrow.

It can work in the following modes:

  1. Using initial seed values, perodically apply a delta to them to mimic vessel movement.
    Seed values and time interval can configured in settings Settings.

  2. Follow a GPX Track. NMEASimulator will walk through each point of the Track at the interval set in the configuration screen.
    Note: If a track has multiple segments, NMEASimulator will concatenate all the segments into one contiguous Track
  3. Manual override allows you to take of control specific values with vessel heading and speed able to be varied using the keyboard_arrow_left keyboard_arrow_right keyboard_arrow_up keyboard_arrow_down keys.
    Additionally all values can be "frozen" at the current value to enable a static steam of data to be output.
NMEA sentences containing the following information are generated:
  • GPS Fix Quality,HDOP, PDOP & VDOP, Number of satellites, Current local time and date.
  • Vessel position, speed and heading data
  • Wind speed and direction
  • Water temperature and depth
  • Waypoint position and name
