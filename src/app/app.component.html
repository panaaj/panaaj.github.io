NMEASimulator is a FREE utilitiy for generating NMEA data streams to devices connected via TCP/IP, WebSocket or Serial port.

NMEASimulator is available for Windows, Mac & Linux platforms and is a driect replacement for the NMEASimulator Chrome App (for which support ends as of June 2020 for these platforms).

It is simple and easy to use tool that mimics vessel movement, engine status, water depth, etc.:

It can work in two modes:

  1. Set initial seed values and then perodically apply a delta to them to mimic vessel movement

  2. Follow a GPX Track. NMEASimulator will walk through each point of the Track at the interval set in the configuration screen.
    Note: If a track has multiple segments, NMEASimulator will concatenate all the segments into one contiguous Track
  • GPS Fix Quality,HDOP, PDOP & VDOP, Number of satellites, Current local time and date.
  • Vessel position, speed and heading data
  • Wind speed and direction
  • Water temperature and depth
  • Waypoint position and name
